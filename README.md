# pucman-ai-game
Классическая аркадная игра Pucman, полностью сгенерированная искусственным интеллектом. Управляйте жёлтым Pucman'ом с помощью стрелок клавиатуры, собирайте белые точки и избегайте разноцветных призраков. Каждая собранная точка приносит 10 очков, столкновение с призраком заканчивает игру, а сбор всех точек означает победу. Игра демонстрирует возможности ИИ в разработке игр и создана в рамках лабораторной работы по продвижению цифровых продуктов.
