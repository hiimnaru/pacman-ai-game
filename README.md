# pucman-ai-game
Pucman-AI-Game — классическая аркадная игра, созданная искусственным интеллектом. Управляйте жёлтым героем стрелками, собирайте точки и избегайте призраков. Полный сбор точек приносит победу, столкновение завершает игру. Проект демонстрирует возможности ИИ в геймдеве для лабораторной работы по продвижению ПО.
